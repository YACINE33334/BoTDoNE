HOW IT WORKS

AUTO MODE: System follows real-world UTC time. Market opens/closes based on strictly defined schedule.

MANUAL TIME: You set a specific time. The system calculates the offset and "freezes" the site to that time logic relative to real time flow.

MANUAL MARKET: Overrides all time logic. If you select "LONDON", London is OPEN green, others are closed. "WEEKEND" labels are removed.
